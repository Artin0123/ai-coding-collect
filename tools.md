# 紀錄各項程式需要的全域規則和 MCP
列出 MCP 的 prompt：
```
列出所有你可以使用的工具以及簡單介紹他們的功能，一行一個，比如：
edit: 編輯檔案
search: 搜尋文字...
回答即可，最後計算共有幾個
```

必備，沒有這個功能就直接刪除：
- 執行終端機命令
- 新增和修改全域規則（系統提示詞）
