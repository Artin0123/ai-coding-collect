# 五子棋遊戲項目簡介

## 項目目標
創建一個完全基於HTML、CSS和JavaScript的五子棋遊戲，支持人機對戰。遊戲需符合標準五子棋規則，包含美觀的UI設計和基本的AI對手。

## 核心要求
- **遊戲規則**：15x15棋盤，5個星位，黑方先行，連五子勝，無重複下棋
- **UI設計**：米黃背景、淺褐棋盤、深褐線條、漸層棋子、置中佈局
- **功能**：點擊下棋、AI自動回應、勝利提示、重新開始按鈕、回合顯示
- **AI策略**：綜合攻防，不輕易讓玩家連線，追求自身勝利

## 技術約束
- 不使用外部庫
- 不使用本地存儲
- 純前端實現
- HTML、CSS、JS三個文件
